# test-effective-mobile
Test exercise with startup simple http server on python with proxy over nginx

Тестовое задание: запуск простого http-сервера на python с проксированием с помощью nginx.

## Предварительная настройка

Запускать рекомендуется под linux, на котором заранее установлен docker и docker-compose. 

## Варианты запуска: 
1) из-под root: дополнительных действий не потребуется.
2) от конкретного пользователя: необходимо предварительно добавить пользователя в группу docker

```sh
$ sudo gpasswd -a $USER docker
```
## Запуск

Перед запуском необходимо скопировать содержимое данного репозитория в рабочий каталог. 

```sh
$ git clone https://github.com/vitalpicnic/test-effective-mobile
```

Для запуска необходимо использовать команду: 

```sh
$ docker-compose up -d
```

## Тестирование 

Проверить работу запущенных приложений можно с помощью команды на хосте, где запущено

```sh
$ curl http://localhost
```
Либо через браузер по адресу http://IP_HOST. Где IP_HOST - адрес хоста/ВМ в сети.

## Дополнительные параметры

Для удобного изменения портов, можно добавить .env файл. Пример содержимого .env: 

```sh
EXTERNAL_PORT=80
INTERNAL_APP_PORT=8080
```

По умолчанию для запуска используются внешний порт 80 для nginx и внутренний 8080 для backend сервера.
